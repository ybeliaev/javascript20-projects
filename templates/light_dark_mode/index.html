Событие change


            const toggleSwitch = document.querySelector('input[type="checkbox"]');

            // Switch Theme Dynamically
            function switchTheme(e) {
                console.log("Switch изменён", e)
            }
            // Event Listener
            toggleSwitch.addEventListener('change', switchTheme);
            В объекте event.target есть свойство checked, которое булево

Теперь можно проверить "чекнут" ли элемент


            function switchTheme(e) {
                console.log(e.target.checked)
            }

Document.documentElement - свойство только для чтения, которое возвращает элемент Element , который является коренным элементом документа document (например элемент <html> для HTML документов).


            function switchTheme(e) {
                if(e.target.checked){
                  document.documentElement.setAttribute("data-theme", "dark");
                  
                } else{
                  document.documentElement.setAttribute("data-theme", "light");
                }
              }

Важно помнить, что в style.css:


            [data-theme="dark"] {
                --primary-color: rgb(150, 65, 255);
                --primary-variant: #6c63ff;
                --secondary-color: #03dac5;
                --on-primary: #000;
                --on-background: rgba(255, 255, 255, 0.9);
                --on-background-alt: rgba(255, 255, 255, 0.7);
                --background: #121212;
              }

Альтернативный метод изменения цвета


            [data-theme="dark"] {
                ...тут старые переменные цветов, но добавить
                filter: invert(1) brightness(0.6);
              }

Фильтр invert() инвертирует все цвета в выбранных элементах

Фильтр brightness() делает элемент ярче или темнее. Значение 0 дает полностью темный элемент. Фильтр invert() делает некоторые элементы очень яркими. Они смягчаются установкой brightness(0.6).
